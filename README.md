# Taller JavaScript HTML Inicio

Proyecto educativo que incluye:

## Características

- **Sistema de Login**: Página de inicio de sesión con validación de email y contraseña
- **Catálogo de Productos**: Sistema de visualización de productos organizados por categorías
- **Categorías**: Pantalones, Gorras, Camisetas, Medias y Zapatos
- **Funcionalidades**:
  - Navegación entre categorías
  - Navegación dentro de cada categoría
  - Visualización de detalles del producto (precio, talla, marca)
  - Cálculo automático de descuento (20%)

## Estructura de archivos

- `inicio.html` - Página de login
- `index.html` - Página de catálogo de productos
- `login.js` - Lógica de validación del formulario de login
- `styles.css` - Estilos CSS
- `img/` - Carpeta con imágenes de productos

## Cómo usar

1. Abrir `inicio.html` en el navegador
2. Llenar el formulario de login (validación: email válido, contraseña mínimo 6 caracteres)
3. Al enviar, se redirigirá a `index.html` con el catálogo de productos
4. Usar los botones para navegar entre categorías y productos

## Imágenes

Las imágenes de los productos están ubicadas en la carpeta `img/`
